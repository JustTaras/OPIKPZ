# Computer Club Database (Course Project)

## Опис
Проєкт розроблений як частина курсової роботи з дисципліни "Бази даних".  
Метою є створення структурованої реляційної бази даних для автоматизації роботи комп'ютерного клубу.

## Документація
- `Vision Document` — опис бачення продукту.
- `BRD & NFR` — бізнес-вимоги та нефункціональні вимоги.
- `ER-Diagram` — схема бази даних.
- `deployment.md` — інструкція з розгортання.

## Структура БД
В системі використовуються такі таблиці:

- **clients**
- **computers**
- **tariffs**
- **sessions**
- **payments**
- **admins**
